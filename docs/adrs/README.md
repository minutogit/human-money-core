# Architecture Decision Records (ADRs)

We use Architecture Decision Records to document significant architectural decisions, including the context, the decision itself, and the consequences.

## Format

We follow a lightweight ADR format. Files should be named sequentially:

`NNN-description-of-decision.md`

Example: `001-use-ed25519-signatures.md`

See [000-adr-template.md](./000-adr-template.md) for the structure of an ADR.
